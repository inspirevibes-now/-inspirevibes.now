## How To Use

1. **Clone the Repository**
   ```bash
   git clone https://github.com/[your-username]/inspirevibes-giveaway-guide.git
   ```
2. **Customize Milestones and Prizes**
   - Open `index.html` and update milestone details, contest rules, and gift items as your campaign evolves.
   - All data is editable within the HTML.

3. **Preview Locally**
   - Open `index.html` with any web browser.

4. **Deploy**
   - Host on GitHub Pages, Netlify, Vercel, or your own server.

## Customization

- **Tabs, colors, images, and data:** Easily change milestone numbers, winner details, prizes, or gift icons by editing the corresponding HTML sections.
- **Instagram links:** Update social handles or contest URLs as needed.

## Tech Stack

- Static HTML
- Tailwind CSS for rapid styling and full responsiveness
- SVG icons (inline, no dependencies)
- Simple JS for tab navigation (no frameworks required)
